## Project setup

```bash
$ npm install
```

# prisma

## Prisma setup

```bash
$ npx prisma generate
$ npx prisma migrate dev
```

# seed

```bash
$ npm run seed
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
